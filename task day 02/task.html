This is a block of text demonstrating the use of a fallback font stack. If the primary font ("Arial") is not available on the user's system, the browser will try "Helvetica". If that is also unavailable, it will default to any available sans-serif font. Using a fallback font stack ensures that text remains readable, regardless of the fonts installed on the user’s device.
